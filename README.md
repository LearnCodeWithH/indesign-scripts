# indesign-scripts
Collection of ID scripts for scanlations

# Style Scripts

* [CloneStyleFromSelectedItemForAllGraphicsInLayer.jsx](CloneStyleFromSelectedItemForAllGraphicsInLayer.jsx) - The style of the selected item will be applied to all graphics present in the currently selected layer.

# Sizing Scripts

* [SizePageToFirstGraphicInLayer.jsx](SizePageToFirstGraphicInLayer.jsx) - (Not working) Sizes a page to the first graphic on the page in the layer. The page size is normalized to 72 dpi resolution, and existing elements are moved to their same relative positions.
* [SizeAllPagesToFirstGraphicInLayer.jsx](SizeAllPagesToFirstGraphicInLayer.jsx) - (Not working) For all pages, sizes a page to the first graphic on the page in the layer. The page size is normalized to 72 dpi resolution, and existing elements are moved to their same relative positions.

# Export Scripts

* [ExportFileToPdfInteractiveAndConvertPagesToPsd.jsx](ExportFileToPdfInteractiveAndConvertPagesToPsd.jsx) - Export project to an Interactive Pdf then invoke Photoshop to convert the Psd to one Psd per page.
